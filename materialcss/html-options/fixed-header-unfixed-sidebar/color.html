Usage

Background Color

To apply a background color, just add the color name and light/darkness as a class to the element.

This is a card panel with a indigo darken-1 class
  • HTML
                                                    
<div class="card-panel indigo darken-1">This is a card panel with a indigo darken-1 class</div>

Text Color

To apply a text color, just append -text to the color class like this:

This is a card panel with dark blue text
  • HTML
                                                    
<div class="card-panel">
    <span class="blue-text text-darken-2">This is a card panel with dark blue text</span>
</div>
